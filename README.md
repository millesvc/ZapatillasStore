# ZapatillasStore

Tienda web demo enfocada en la venta y exhibición de zapatillas, con una propuesta visual moderna, dinámica y orientada a marcas urbanas y sneakers.

## Descripción

ZapatillasStore fue creada como una demo de e-commerce visual, pensada para mostrar productos de manera atractiva, destacar categorías y ofrecer una experiencia moderna al usuario.

Este proyecto busca simular una tienda online con identidad fuerte, estilo urbano y enfoque comercial.

## Características

- Catálogo visual de zapatillas
- Diseño responsive
- Estética moderna y urbana
- Secciones organizadas por productos o categorías
- Interfaz pensada para venta y exhibición
- Base escalable para futuras funciones e-commerce

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript

## Objetivo

Demostrar cómo podría verse una tienda online de zapatillas con una presentación visual potente, profesional y preparada para evolucionar hacia una experiencia más completa de venta.

## Estado del proyecto

Demo en evolución, con mejoras continuas en diseño, contenido y experiencia de usuario.
